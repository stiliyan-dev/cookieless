# Cookieless Promo Video Storyboard

## Goal

Show that Cookieless keeps browsing quiet, simple, and local-first without turning the popup into a power-user dashboard.

## Scene 1

- Website loads with a cookie banner
- Cookieless badge is visible in the toolbar
- On-screen text: `Quiet browsing by default`

## Scene 2

- Popup opens
- `Recommended`, `Hide only`, and `Stronger reject` are visible
- On-screen text: `Three clear browsing styles`

## Scene 3

- User clicks `Pause on this site`
- Page is left alone and popup status changes to paused
- On-screen text: `Fast recovery when a site misbehaves`

## Scene 4

- User clicks `Report broken site`
- First click shows a short disclosure and consent checkbox
- Report is sent to support only after explicit acknowledgment
- Later click shows `Reported` with no redirect
- On-screen text: `Optional report, sent only when you choose`

## Closing Frame

- Cookie-based icon and wordmark on screen
- Closing line: `Cookieless. Quiet cookie handling for Chrome.`
